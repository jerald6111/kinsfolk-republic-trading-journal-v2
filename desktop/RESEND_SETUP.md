### Email Not Arriving?

1. **Check Spam Folder**
   - Emails from `onboarding@resend.dev` might go to spam initially
   - Mark as "Not Spam" to fix this

2. **Add to Safe Senders**
   - Add `onboarding@resend.dev` to your contacts
   - Create email filter to move to inbox

3. **Wait a Moment**
   - Emails usually arrive within seconds
   - Can take up to 1-2 minutes during peak times

4. **Check Email Address**
   - Verify you entered correct email
   - No typos in the address

### Error Message?

If you see an error:
1. Check your internet connection
2. Try again in a few minutes
3. The system will download the backup file as fallback
4. You can manually email the file to yourself
